CSS - Cascading Style Sheets

CSS is used to enhace the presentation/look of a web page , making it more user friendly(enhancing user experience) 


        selector {
            property: value;
            property: value;
            color: red;
            font-stye: italic;
        }

        h2{
            color: green;
        }

{} - declaration block - a group of declarations(property+value) that will apply to the selected element

Selector - identify an element or a group of elements to aplly some sytyles

Types of Selectors

  • Element Name / Tag name Selector

                    li{
                    color: red;
                }

  • Class Selector 

                    .specialelements{
                    color: pink;
                }

  • ID selector

                    #someel{
                    background: green;
                }
  • Pseudo class selectors
  • Pseudo element selectors
  • Combinator selectors -- class and class, class and name ....

    • -- Descendant , Child , Adjacent Sibling ...

Where To Put the CSS

  • Inline CSS - using style attribute
  • Internal CSS -- within the html document, in a style element
  • External CSS -- writing styles in a .css file linked to a html page or multiple html pages
